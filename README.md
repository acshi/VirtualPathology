# VirtualPathology

## Setup

Open the project in Unity 5.50b
Open scene "main_vr"
Install the packages "SteamVR" "VRTK" from the Unity Asset Store

Install Steam/Steam VR (outside of unity) and run it
Run room setup

Run the application

##Hardware Recomendations

Nvidia 1080
32 GB RAM
SSD for storing data
